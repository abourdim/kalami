What Kalami does
كل ما تحتاجه · Everything You Need
🎙️
Arabic Speech Input
Web Speech API with 5 dialect modes — MSA/Gulf, Egyptian, Maghrebi, Levantine, Iraqi. Editable transcript to fix recognition errors.
🇫🇷🇬🇧
Dual Translation
Arabic → French and Arabic → English in parallel via MyMemory free API. Side-by-side panels, newest translation on top.
⚡
Near-Live Mode
Translates and speaks each phrase chunk ~2–3 seconds after you say it. Audio queue engine prevents voice overlap.
⌨️
Arabic Virtual Keyboard
Full 28-letter keyboard with numerals ١–٩, punctuation ، ؟, backspace, word-delete, and a direct Translate key.
🧠
Translation Memory
Repeated phrases translate instantly from in-session cache — no API call, no lag. ⚡ badge confirms cache hit.
↩️
Back-Translation Verify
Translates FR/EN back to Arabic under each entry so you can spot-check accuracy. Toggle on/off per session.
🔊
Voice Playback
Web Speech Synthesis for both languages. Pick any installed voice, adjust speed 0.5× – 2.0×, choose FR / EN / Both.
📑
Export Sessions
Download full bilingual transcripts as .TXT or print-ready PDF with Bismillah header, Arabic and both translations.
⭐
Star · Search · Fullscreen
Bookmark important entries, filter history by keyword, expand any panel to fullscreen for meetings or presentations.
